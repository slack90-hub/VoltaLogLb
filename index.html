Questions customers ask.

Short answers for buyers comparing VoltaLog with manual spreadsheets or generic business software.

Is VoltaLog for electricity billing only?

Yes. It is specialized for customer subscriptions, meter readings, tariffs, invoices, payments, receipts, and reports used by generator and electricity billing offices.

Does it need internet?

The desktop app is designed for local Windows operation. Internet is mainly useful for downloading releases, updates, and support.

Can I export data to Excel?

The system is designed to be Excel-friendly for reports and operational lists, so offices can still share and review data easily.

Can customers receive printed documents?

Yes. The workflow supports customer-facing invoices, receipts, statements, and report output.

Is my source code public?

No. This is only the public product and release page. The application source code remains private.

Where do I download it?

Use the releases section of this public repository. Each package is created by the private build workflow and published here for customer access.
