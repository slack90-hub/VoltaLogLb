Questions customers ask.

Short answers for buyers comparing VoltaLog with manual spreadsheets or generic business software.

Is VoltaLog for electricity billing only?

Yes. It is specialized for customer subscriptions, meter readings, tariffs, invoices, payments, receipts, and reports used by electricity billing offices.

Does it need internet?

The desktop app is designed for local Windows operation. Internet is mainly useful for downloading releases and updates.

Can customers receive printed documents?

Yes. The workflow supports customer-facing invoices, receipts, statements, and report output, including Arabic copies where configured.

Where do I download it?

Use the releases section of this public repository. Each ZIP package is created by the private build workflow and published here for customer access.
